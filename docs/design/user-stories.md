| ID        | Category         | Priority | User Story                                                                                                                                                                                                                             | Acceptance Criteria                                                                                                                                                                                                                                                                                                                                                                   |
| --------- | --------- | --------- | --------- | --------- |
| **US-01** | Image Processing | High     | As a user on mobile, I want to upload a photo from my camera roll and have a face automatically detected so that I don't have to manually crop or select the face; it just works.                                                      | • Face bounding box appears within **1 second** of image upload.<br>• Accuracy **>85%** on well-lit frontal faces.<br>• If detection fails, a clear inline error message explains why.                                                                                                                                                                                                |
| **US-02** | Frontend         | High     | As a user, I want to browse and select from a library of popular meme templates so that I can find the right meme format without knowing its name.                                                                                     | • Template grid loads within **1.5 seconds**.<br>• At least **20 templates** available at launch.<br>• Search returns results in **<500ms**.<br>• Templates display name and preview thumbnail.                                                                                                                                                                                       |
| **US-03** | Image Processing | Medium   | As a user, I want to see my face composited into the selected meme template with correct proportions so that the result looks intentional, not stretched or off-center.                                                                | • Face is positioned in the designated face region of the template.<br>• No visible distortion on faces between **50×50px** and **500×500px**.<br>• Result matches the template’s face-region bounding box within **10px**.                                                                                                                                                           |
| **US-04** | Image Processing | Medium   | As a Reddit or Discord user, I want the final meme to be a small enough file size to send over Discord or Reddit without looking super pixelated so that I can quickly share the image after generation without compressing it myself. | • Final image size is **≤10 MB**.<br>• Image remains clear and understandable (not noticeably pixelated).                                                                                                                                                                                                                                                                             |
| **US-05** | API Integration  | High     | As a user, I want to share the completed meme directly to my contacts or social apps so that the joke lands while everyone is still paying attention.                                                                                  | • Web Share API opens within **300ms** of tapping Share.<br>• Fallback clipboard copy works on all browsers.<br>• Image exports at a minimum width of **800px**.<br>• Sharing works without requiring account creation.                                                                                                                                                               |
| **US-06** | AI & Prompting   | P0       | As a user in extra roast mode, I want to generate a completely new meme image using only my friend's face so that the meme is fresh and not limited to existing templates, providing maximum surprise factor.                          | • AI generation completes in **<8 seconds**.<br>• Prompt directs the model toward absurdist, deep-fried, or cursed aesthetics.<br>• Generated image embeds the uploaded face and keeps the person recognizable.<br>• Output is downloadable at **1024×1024px** minimum.<br>• If generation fails or triggers a safety filter, show a friendly retry prompt instead of a blank screen. |
| **US-07** | AI & Prompting   | P0       | As a user in friendly roast mode, I want to generate a kind, funny meme that I could send to my grandma or school group chat so that I don't accidentally send something mean or inappropriate.                                        | • Generated content passes a **PG-13** filter (no explicit content, slurs, or targeted harassment).<br>• Tone prompt instructs the model toward affectionate humor.<br>• User can flag a result if it feels inappropriate or off-tone.                                                                                                                                                |
| **US-10** | API Integration  | P1       | As a user on a slow mobile connection, I want the app to remain usable even if the AI API is slow or unavailable so that I can still make a basic meme when AI features are down.                                                      | • Normal mode (face swap) works fully offline using cached templates.<br>• AI failures show a clear error message and offer normal mode as a fallback.<br>• No blank screens or unhandled errors.                                                                                                                                                                                     |
