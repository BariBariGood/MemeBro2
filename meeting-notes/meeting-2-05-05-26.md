Ideas & Brainstorm
- Discussed the overall vision for MemeBro — a web app that lets users upload a photo and generate memes using AI
- Reviewed the core features: meme template selection, face swapping, AI caption generation, and text editing
- Looked at references like ImgFlip and GIPHY for inspiration
- Decided to build as a web app (Cloudflare Workers + static frontend) for accessibility across devices

Questions & Decisions
- Split the team into two groups: Frontend (UI/editor) and Backend (API/worker)
- Frontend team: responsible for the template selection screen, editor canvas, and text editing
- Backend team: responsible for the Cloudflare Worker, API routes, and AI integrations
- Decided to use GitHub for version control with feature branches per task
- Each feature will live on its own branch and be reviewed before merging into main

Plan Going Forward
- Set up the GitHub repository and invite all members
- Each member to clone the repo and get familiar with the project structure
- Frontend team to begin working on the template selection and editor layout
- Backend team to begin researching AI providers for face swapping and caption generation
